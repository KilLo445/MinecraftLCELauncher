# Minecraft Legacy Launcher
Simple .bat launcher for Minecraft Legacy

Automatically enters Name, IP, and Port arguments based on input. Leave them blank for defaults.

## Installation
1. Download [Minecraft.Client.Launcher.bat](https://github.com/KilLo445/MinecraftLCELauncher/raw/refs/heads/main/Minecraft.Client.Launcher.bat) ([Mirror](https://files.killo.club/software/Minecraft.Client.Launcher.bat))
2. Place it next to `Minecraft.Client.exe`
3. Open it, enter your Name, IP*, and Port*
4. Confirm and play

*Optional, leave blank for default
